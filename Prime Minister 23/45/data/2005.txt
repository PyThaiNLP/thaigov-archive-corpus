คำกล่าวของนายกรัฐมนตรีในพิธีเปิดการประชุมเชิงปฏิบัติการเรื่อง

 ยุทธศาสตร์เพื่อการแข่งขันของเศรษฐกิจไทยภายใต้สถานการณ์ใหม่ของเศรษฐกิจโลก

ณ ตึกสันติไมตรี ทำเนียบรัฐบาล

นรม:     ผมขอใช้เวลา 15 นาทีเพื่อซักซ้อมความเข้าใจให้ตรงกัน ถ้าผมมีอะไรเข้าใจไม่ตรงกัน ขอให้บอกผมด้วย  หัวข้อที่จะพูดกันวันนี้คือ ยุทธศาสตร์เพื่อการแข่งขันของเศรษฐกิจไทยภายใต้สถานการณ์ใหม่ของเศรษฐกิจโลก ซึ่งเป็นหัวข้อชัดเจนมากเพราะว่า เรากำลังถามตัวเองว่า เราอยู่ในจุดไหนของสถานการณ์เศรษฐกิจโลกที่กำลังเปลี่ยนไป เพราะไม่มีอะไรที่ไม่เปลี่ยน เพราะเราถือว่าเศรษฐกิจ

ก็เหมือนสิ่งมีชีวิตที่ต้องการมีการเปลี่ยนแปลง มีสภาวะการณ์แวดล้อม ( environment ) เราจึงต้องปรับตัว เรากำลังถามตัวเองว่า เราต้องปรับตัวมากน้อยเพียงไหนกับสิ่งที่เปลี่ยนไป ถ้าจะถามอย่างนี้ก็เหมือนกับว่า เรากำลังถามตัวเองว่า ตัวเราเองที่เป็นมนุษย์นี้ เรารู้ว่าเราอยู่ในประเทศไทย ซึ่งมีฤดูร้อน หนาว ฝน อันนี้เป็นภาพมหภาค ที่จะรู้ว่า ฤดูร้อนควรจะทำอย่างไร ฤดูฝนและฤดูหนาวควรจะทำอย่างไร แต่มีเหตุการณ์จุลภาคที่เกิดขึ้นได้ อาทิเช่น น้ำท่วม โรคระบาด เราควรต้องทำอะไร ในภาวะนั้น เรามาประชุมตกลงกัน เราจะเตรียมตัวอย่างไร แต่ในวันนี้เช่นกัน เราจะต้องกลับมาถามก่อนว่าในมหภาค ในภาพกว้างของโลกนี้มีอะไรเปลี่ยนแปลงไป แล้วในเหตุการณ์เฉพาะอย่าง เฉพาะเรื่อง ที่มีผลกระทบกับเรา

มีอะไรเปลี่ยนไป แล้วเราได้รับมือกับความเปลี่ยนแปลง ( cope with change ) เหล่านี้คือสิ่งที่ต้องถาม ต้องมีการซักซ้อม ถ้าเราไม่ซักซ้อม ไม่ถามกันตรงนี้แล้ว เราจะ สรุป ( end up ) ว่าเราไม่รู้ปัญหา แสดงว่า เราไม่รู้ปัญหาจริง เรื่องนี้เรื่องใหญ่ที่เราจะต้องปรับตัว ถามตัวเอง ผมจะซักซ้อมสักนิดหนึ่งว่า ในความรู้สึกผมนี้ ผมว่ามีอะไรบ้าง เพื่อในความรู้สึกของท่าน ท่านจะได้รู้สึกว่ามันมีอะไรมากกว่าที่ผมพูด หรือสิ่งที่ผมพูดนั้นไม่จริง เราจะได้เอามาเป็นข้อมูล ( input ) ในการมองสิ่งที่นำเสนอต่อไปนี้

            ในภาพกว้างเรื่องแรก ผมคิดว่า วิธีคิดของคนในโลกนี้เปลี่ยนไป เพราะว่าการมีส่วนร่วมของคนเป็นหัวใจที่สำคัญ ที่ผมเคยพูดตลอดเวลาว่า  คนจะทำอุตสาหกรรมอะไร เมื่อก่อนนี้ เราไม่เคย

สนใจลูกค้า เราสนใจแต่ว่าเราจะผลิตอะไร  ราคาเท่าไหร่ ก็กำหนดเอาเอง แต่วันนี้ไม่ใช่ วันนี้เราต้อง

สนใจลูกค้า ลูกค้าจะต้องเป็นคนบอกว่า ต้องการอะไร สีสรรอย่างไร และราคาเท่าไหร่จึงจะซื้อ วิธีคิดมันเปลี่ยนไป มันกลับจากภายในสู่ภายนอก เป็นภายนอกสู่ภายใน ( inside out เป็น outside in ) นั่นคือ อันแรกที่จะถามว่าเรารู้หรือเปล่า เราตระหนักหรือเปล่า เราในที่นี้คือทั้งรัฐบาลและภาคเอกชน และวันนี้ dynamism สูงมาก มีพลวัตรสูงมากด้วย ICT หรือ Information and Communication Technology ซึ่งทำให้ทุกสิ่งมีพลวัตรสูงมาก สินค้า ( mass production ) วันนี้ วงจรชีวิตสั้นมาก ( life cycle ) สั้นมาก และคำว่า mass มัน mass จริงๆด้วย และด้วยเทคโนโลยีทำให้ mass มากกว่าเดิม ฉะนั้น เมื่อมัน mass มากกว่าเดิม life cycle มันเลยสั้นขึ้น มันเข้าไปสู่ mass customization เกิดขึ้น ทุกอย่างเป็นวิกฤติและเป็นโอกาสในตัวมันเอง เมื่อก่อนนี้ mass production เราไม่ทัน technology เราสู้เขาไม่ได้เป็นวิกฤติ เวลานี้มาสู่ mass customization เป็นวิกฤติและเป็นโอกาสด้วย เพราะเรามี heritage talent มี culture มันก็จะเป็นโอกาส เพราะสิ่งที่เราพูดไป เราจะเห็นวิกฤติและโอกาสตลอดเวลา โอกาสนี้ หากปล่อยไปนิดเดียว จะกลายเป็นวิกฤติ เพราะทุกวิกฤติมันเจาะให้เกิดโอกาสได้ นี่คือสิ่งที่ผมอยากจะย้ำว่า วันนี้ท่านทั้งหลายจะต้องคิดแบบบูรณาการ ให้มากๆ คิดความเชื่อมโยงให้มากๆ เพราะคำว่า globalization ที่เผชิญพวกเราทุกวันนี้ มันคือ การเกิดภาวะ borderless world เกิดภาวะความเชื่อมโยงของโลก เกิดภาวะที่มี free flow ของ human resource ยกตัวอย่างง่ายๆ แรงงานพม่าที่อพยพเข้าไทย เพราะเศรษฐกิจไทยดีกว่าพม่า แรงงานไทยไปมาเลเซีย สิงคโปร์ และไต้หวัน เพราะเศรษฐกิจเขาสูงกว่าเรา แรงงานสิงคโปร์ และแรงงานไต้หวันไป สหรัฐฯ จะเห็นว่าที่ไหนให้โอกาส

( opportunity )ที่นั่นคนไป เพราะเกิดการไหลเวียน ( free flow ) ขึ้นแม้ว่า จะไร้พรมแดน

( borderless )บ้าง มีพรมแดน (ไม่ borderless) บ้าง มันก็ free flow  เป็น free flow ของเงิน เงินไหลเข้าออกเร็วมาก จนบางครั้งเป็นอันตรายต่อประเทศที่ไม่มีระบบ monitoring ที่ดีอย่างไทยที่ได้ประสบไปแล้ว free flow trade หรือ goods เกิดขึ้นมา นี่คือสิ่งที่เราจะต้องติดตามต่อ เราเข้าสู่ยุคของ knowledge base ซึ่ง product derived from brain มีราคาดีกว่า product derived from skill หรือ from human labour เพราะฉะนั้น สมอง ( brain ) เป็นหัวใจสำคัญที่สุดในโลกยุคใหม่ ถามว่าเราเตรียมความพร้อมรึยัง เรามีinnovation ใหม่หรือไม่ ที่จะอยู่กับโลกที่เป็น knowledge base development เพื่อ ค้นหาความรู้ใหม่ๆ ( seek new knowledge ) ซึ่ง การค้นคว้าและวิจัย

( Research and Development ) ที่กำลังมาแรงในยุคปัจจุบันนี้คือ biotechnology  พวก lifestyle chemistry เพราะว่า ถึงแม้ว่าตอนนี้ โลกกำลังใช้ประโยชน์มากด้าน IT หรือ ด้านฟิสิกส์ แต่วันนี้ new knowledge สำหรับฟิสิกส์จะเริ่มต้นเพราะว่าทฤษฎี ( Theory ) มันไปไกลมาก เพราะกำลังมีการ

แสวงหาทฤษฎีในแนวใหม่ ในแนวทางด้านเคมี ด้าน bio-lifestyle  และสิ่งเหล่านี้ทฤษฎีจะเชื่อมเข้าหากัน จะเป็นความรู้ใหม่ที่เกิดขึ้น นี่เป็นสิ่งที่จะต้องติดตามและถามว่ามีความพร้อมหรือยัง อันนี้คือคำถาม โลกกำลังสนใจเรื่อง ( health conscious ) ทุกคนสนใจในเรื่องสุขภาพ กลัวตาย กลัวป่วย อยากมีอายุยืนนาน อยากมีสุขภาพที่ดี นี่เป็นโอกาสของไทยหรือไม่ คือสิ่งที่ กำลังถามท่าน เขากำลังเป็นห่วงถึง การตัดต่อทางพันธุกรรม ( GMO ) เรื่องของสารเคมีตกค้าง นี่เป็นโอกาสหรือไม่ หรือจะปล่อยให้เป็นวิกฤติ นี่คือสิ่งที่ถาม

            เรื่อง internet ทำให้โลกมีการเชื่อมโยงกันอย่างมากมาย  internet เป็นวิกฤติและเป็นโอกาส internet ทำให้คนตัวเล็กมีโอกาสสูงขึ้น เพราะด้วยระบบของ e-commerce ระบบwebsite ทำให้มี window โชว์สินค้าของคนตัวเล็ก และ internet กำลังบอกว่าคนที่เป็น broker ถ้าไม่ เพิ่มมูลค่า ( add value ) ก็ไม่สามารถยืนอยู่ในโลกยุคใหม่ได้ เพราะ broker ต้อง add value นี่คือสิ่งที่เกิดขึ้น ถ้าถามว่าวันนี้ เราไม่มีความเก่า ไม่มีความเป็น broker แต่เป็นคนตัวเล็กเยอะ SMEs มาก community business เยอะ นี่คือโอกาส แต่ถ้าเราไม่สามารถขยับตรงนี้ทัน ก็เป็นวิกฤติ เหตุการณ์ 11 กันยายน ที่สหรัฐฯ นี่คือ Micro นี่คือจุลภาคที่เปลี่ยนแปลงบางอย่างของโลก เช่น ท่องเที่ยวได้เปลี่ยนไป คนไปเที่ยวสหรัฐฯน้อยลงแน่นอน เพราะความไม่สะดวก เพราะ สหรัฐฯต้องเคร่งเครียดเรื่องความปลอดภัยมาก ความกลัวเรื่องต่างๆแน่นอน เป็นวิกฤติของเราคือ นักท่องเที่ยวที่ออกจากสหรัฐฯก็น้อย แต่ก็เป็นโอกาสของเรา คือคนที่จะไปสหรัฐฯก็หาที่ไปใหม่ เป็นสิ่งที่เราต้องดู ต้องติดตามและต้อง tab ให้ได้ นี่คือสิ่งที่เราจะต้องบอกกันให้รู้และเรื่องของการลงทุน เมื่อก่อนบอกกันว่า สหรัฐฯเป็นสถานที่ที่ปลอดภัย ( safe place ) ที่สุดแล้ว แต่วันนี้ คนเริ่มมีความสงสัยว่า จะต้องมีการกระจายการลงทุนไปภูมิภาคอื่นหรือไม่ นี่ก็เป็นทั้งโอกาสและวิกฤติ

            โลกเริ่มเกิดการก่อตัวรวมตัว เพื่อให้เกิด economy of scale เรื่องของการขับเคลื่อนเขตการค้าเสรี ( FTA )เป็นคู่ๆ เพื่อให้เกิด economy of speed เกิดขึ้นในโลกนี้ ถามว่าเราใช้ประโยชน์

( benefit ) ตรงนี้หรือไม่ เราต้องเตรียมความพร้อมแค่ไหน  เราจะขึ้นไปอยู่บน FTA ไหวหรือไม่ การที่จีนเข้าสู่ WTO เป็นโอกาสหรือเป็นวิกฤติ นี่คือ ตลาดของเราหรือเปล่า การที่เรา initiate ACD เพื่อให้คนเอเชียเปิดตลาดเอเชียเข้าหากัน แทนที่เราเอเชียจะแย่งกันผลิตเพื่อกด ( dump ) อยู่ในตลาดนอกเอเชีย เราควรจะปรับตัวอย่างไร เพราะทุกวิกฤติมีโอกาส ทุกโอกาสถ้าปล่อยให้หลุดลอย จะเป็นวิกฤติ

            การกีดกันทางการค้า ( trade barrier ) และ protectionism กลับมาใหม่ เมื่อก่อน protectionism ออกมาในรูปของภาษี แต่วันนี้ non-tariff-barrier เป็นเรื่องใหญ่ และยากกว่าเรื่อง tariff barrier ด้วยซ้ำ แล้วรูปแบบต่างๆเริ่มออกมามากมาย ซึ่งถามว่าเราจะตั้งรับอย่างเดียวหรือเปล่า แล้วจะรุก จะรุกอย่างไร นี่คือสิ่งที่ต้องถาม แน่นอน ถ้าเราเข้าใจคำว่า international standard และ efficiency เราจะมีปัญหาน้อย เมื่อไหร่ถ้าเราบอกว่าเราจะต้อง ขายของถูกๆ เหมือนคนลดน้ำหนักที่ลดจนไม่มีอะไรจะลดแล้ว จนต้องตัดแขนตัดขาเพื่อให้น้ำหนักเบาลง ในที่สุดเราก็จะจนกันอยู่อย่างนี้  นี่คือสิ่งที่จะต้องถามตนเอง และถามว่าเราจะยืนอยู่ตรงไหน เราจะปรับตัวอย่างไร กับสิ่งที่เปลี่ยนไปตอนนี้ นี่คือคำถามที่เราต้องตอบ ในระยะสั้น ระยะกลาง ระยะยาว ระยะ 3ปี 7 ปี และ 15 ปีข้างหน้าจะทำอย่างไร คือสิ่งที่เราจะต้องทำทุกๆอัน ต้องทำพร้อมๆกัน ไม่ใช่ระยะ 15 ปีข้างหน้ารออีก 10 ปีค่อยทำ มันต้องทำตั้งแต่วันนี้ เพราะมันใช้เวลา ยกตัวอย่างง่ายๆ เรื่องแรก knowledge base economy ระบบการศึกษาเราเสียไปหนึ่ง generation เด็กไทยจบมาคิดไม่ค่อยเป็น บูรณาการไม่รู้ เชื่อมความคิดความเชื่อมโยงไม่ถูก นี่คือจุดอ่อนของการศึกษาเรา ไม่ค่อยกล้าคิด ไม่กล้าที่จะล้มเหลว(dare to fail) นี่คือจุดที่เราต้องใช้เวลา ในการฟื้นมันขึ้น จะถามว่ามันเป็นวิธีทั้งหมดหรือเปล่า ก็ไม่ถึงขนาดนั้น เพราะว่าระบบของ information technology ทำให้ flow ของ information ที่เข้าสู่สมองเด็กรุ่นใหม่นี้ มีมากกว่าสมัยเราเยอะ แม้ระบบการศึกษาจะไม่ดี แต่การรับ information  มากกว่ารุ่นพวกเรา นั่นคือสิ่งที่เป็นประโยชน์อยู่ แต่ว่า education ต้องใช้เวลานาน ต้องแก้ไข

            เรื่องของนวัตกรรม ( innovation ) ที่เราอยากได้ idea ใหม่ๆ ความรู้ใหม่ๆ ได้ break through ของเทคโนโลยีใหม่ๆ นั้น ก็เรื่อง R & D ซึ่งเราส่งเสริม แบบไม่ส่งเสริมมาตลอด วันนี้จะต้องถามว่าจะส่งเสริมอย่างไรจึงจะถูก ต้องถามก่อน ต้องตั้งต้นก่อน เพราะสิ่งที่ส่งเสริมมาเดิมนั้น มันดูเหมือนส่งเสริม แต่ไม่ได้ส่งเสริม อันใหม่ก็ต้องมาถามว่าเราจะส่งเสริมอย่างไร ที่มันจะส่งเสริม ส่งเสริมแบบที่เรา afford ไหว แต่บางคนที่ไม่เข้าใจ ไม่เห็นคุณค่าเหล่านี้ การส่งเสริม 100 ล้านก็ว่ามากไปแล้ว  แต่ถ้าเห็นคุณค่า 2-3 หมื่นล้านก็ธรรมดา อยู่ที่ว่าเราเห็นคุณค่าและ ได้ผลแค่ไหน ซึ่งเป็นเรื่องของ R & D

            มาดูด้านโครงสร้าง ( structure ) ของเรา คือภาครัฐบาลมีหน้าที่อำนวยความสะดวก

( facilitate ) ให้ทุกสิ่งทุกอย่างมันดีขึ้น ฉะนั้นต้องกลับมาแก้ไขด้านการบริหาร ( management ) ของภาครัฐ ภาครัฐมีหน้าที่ facilitate ให้ภาคเอกชนและภาคประชาชนเข้มแข็ง เราก็ต้องกลับมาดูว่ากระบวนการบริหารที่เป็นระบบไม่มีเจ้าภาพ ระบบที่ไม่ตระหนักคำว่า efficiency และเป็นระบบที่ใช้เทคโนโลยีต่ำ มีกฎหมายเยอะ มีกติกามาก จนทำอะไรไม่ได้ ตรงนี้ต้องแก้ ต้องใช้เวลาแน่นอน ในด้านของ human resource development ต้องพัฒนาเข้าสู่ knowledge base แน่นอน เพราะไม่ฉะนั้น จะมี clerk หลายชั้น ตั้งแต่ชั้นซี 3 จนถึงซี 10 และดีไม่ดีจะเป็น clerk ระดับรัฐมนตรี และรอคณะรัฐมนตรีตัดสินอย่างเดียว ซึ่งระบบนี้ต้องได้รับการแก้ไขแน่นอน

            อีกเรื่องคือ เรื่องการเอาผลประโยชน์ของชาติเป็นที่ตั้ง หรือ nationalism อันนี้เราอ่อน (weak)กันมาก ซึ่งเป็นปัจจัยสำคัญของความสำเร็จ ( key ingredient for success ) ของความสำเร็จของระบบเศรษฐกิจแบบทุนนิยม   ผมเห็นตัววิ่งใน CNN ปรกฏว่าสภาของหสรัฐอเมริกากำลังพิจารณาแก้กฎหมาย ที่มองว่าบริษัทอเมริกาที่เอา head quarter ไปตั้งที่ ประเทศที่เป็นสวรรค์ด้านภาษี (tax heaven country) บอกว่าเป็นพวกไม่รักชาติ ( non-patriotic )  เห็นมั้ยครับว่าอเมริกากำลังจะแย่เพราะมีบริษัทอเมริกาเลี่ยงไปตั้งสำนักงานใหญ่ ( head quarter ) ทางทะเบียน ทางกฎหมาย ที่ปานามา ที่ British Virgin Island บ้าง ส่วนสำนักงานใหญ่ ( Head Quarter ) จริงๆตั้งอยู่ที่อเมริกาเพื่อเลี่ยงภาษี อเมริกาเรียกพฤติกรรมเช่นนั้นว่า non-patriotic เขาจึงแก้กฎหมาย นี่คืออเมริกาที่ซึ่งเป็นประเทศที่มีเสรีภาพสูงมาก เป็นประเทศที่ให้โอกาสคนทั่วโลก ไปช่วยกันสร้างอเมริกาให้เข้มแข็ง เราต้องสร้าง เครื่องจักรกลทางเศรษฐกิจ (new economic engine) ต้องสร้างเครื่องยนต์ทางเศรษฐกิจใหม่ๆ และ ปรับปรุงยกเครื่อง (overhaul) เครื่องยนต์เก่าๆ ที่ยังพอ overhaul ได้อยู่ ถ้าไม่เช่นนั้นเราก็ไม่สามารถจะอยู่บนโลกที่เปลี่ยนไปได้อย่างดี 

            เรื่องของการสร้าง entrepreneur คือผู้ประกอบการรุ่นใหม่ เพราะผู้ประกอบการรุ่นเก่าๆ

ส่วนใหญ่จะเป็นผู้ประกอบการที่มีการศึกษาไม่สูง ฉะนั้นภาคธุรกิจของเราจึงไม่ค่อยเข้มแข็ง หรือ

เข้มแข็งแบบไม่เข้มแข็ง เพราะคำว่าประสิทธิภาพ (efficiency) ที่ต้องอาศัยความเป็นมืออาชีพ (professionalism) ไม่ค่อยเพียงพอ ดังนั้น ผู้ประกอบการใหม่ๆที่การศึกษาที่สูงขึ้นนั้นจะต้องได้รับการสร้างในช่วงนี้

            การท่องเที่ยว แน่นอนครับในโลกข้างหน้าเราจะพูดถึงดุลการค้า  (trade balance) เพราะทุกประเทศไปถึงก็จะพูดว่าคุณ ( I ) เสียเปรียบการค้าคุณ ( you ) และ คุณต้องสั่งซื้อมากขึ้น เมื่อพูดถึง trade balance ถามว่าเงินสกุลแข็ง ( hard currency ) เราจะหา (earn) จากไหน การท่องเที่ยวครับ ฉะนั้น การท่องเที่ยวจะต้องมีการปรับปรุงยกเครื่อง ( overhaul )  ครั้งใหญ่ให้เกิดการพัฒนาท่องเที่ยว และต้องมี synergy ของประเทศเพื่อนบ้าน อย่างเป็นรูปธรรม เพื่อให้แม่เหล็กของเราซึ่งใช้งานมามากอาจจะเสื่อมไป และได้เพิ่มพลังแม่เหล็กในการดึงดูดมากขึ้น

            ยุทธศาสตร์การส่งออก (export strategy) ต้องพอเพียง การทดแทนการนำเข้า (import substitution) ต้องรักษา ( keep ) เพื่อให้เงินสกุลแข็ง ( hard currency )ยังอยู่ประเทศไทย มันเหมือนกับการค้า ถ้าเราได้สุทธิ ( net) แล้วไปบวกก็มีกำไร ( profit ) ซึ่งโดยพื้นฐาน ( fundamental ) ของการทำธุรกิจคือต้องมีกำไร ( profit )  และมีพื้นฐาน ( fundamental )ของการบริหารเศรษฐกิจ(economy)คือเราต้องหาเงินสกุลแข็ง ( earn hard currency) เราต้องมี ส่วนเกิน (surplus) เราไม่ใช่มีเงินฝาก (deposit)  นั่นคือหลักที่เราต้องทำ สิ่งที่เราต้องช่วยกันในโครงสร้างของภาคเอกชนที่ต้องช่วยกันในภาคเอกชนก็คือ คำว่าประสิทธิภาพ ( efficiency ) ในภาคของเอกชน ( corporate ) ของเรานั้น วันนี้ ผมคิดว่าเราจะต้องให้เขาแข็งแรงขึ้น บางที่แข็งแรงบางที่ยังไม่ค่อยแข็งแรง  บางที่ยังโตด้วยหนี้ ที่เราล่มกันมาครั้งที่แล้วเพราะหลายบริษัทโตด้วยหนี้ ไม่ได้โตด้วยประสิทธิภาพ ( efficiency ) หรือ พื้นฐาน (fundamental) ของธุรกิจ ฉะนั้นเราต้องช่วยให้ความรู้เสริมสร้างภาคเอกชนให้เข้มแข็งด้วย ส่วนปัญหาย่อยๆเราก็ต้องมาดูกันว่าเราต้องปรับภาค ( sector ) ต่างๆ ให้สอดคล้องกับการเปลี่ยนแปลงต่างๆของโลก ปรับภาคเกษตรเรื่องทิศทางต้องชัดเจน และเรื่องของการเพิ่มผลผลิต โลกที่เปลี่ยนไป ที่ต้องการการผลิตตามความต้องการของลูกค้า ( mass customization )ภาคอุตสาหกรรมต้องมีการปรับเปลี่ยนอย่างไร

            เทคโนโลยีการสื่อสารและสารสนเทศ ( ICT ) เราจะเอาอย่างไร ซึ่งมันมี 2 ส่วน ส่วนหนึ่งก็คือ ในฐานะที่เราจะเป็นนักประดิษฐ์คิดค้น ตรงนั้นเราค่อนข้างจะตามเขาไม่ทัน แต่ก็ไม่ได้หมายความว่าไม่ควรจะทำ แต่คู่ขนานกันไปคือ การซื้อ เทคโนโลยี และการใช้ให้เกิดประโยชน์ ผมยกตัวอย่างเมื่อคืนนี้ ดู CNN เห็นข่าว NOKIA ตกลงขาย software ให้ซีเมนส์เอาไปใช้ เห็นมั้ยครับว่า เป็นคู่แข่งกัน แต่ขาย software เพราะซีเมนส์ ( Siemens ) ต้องการโตเร็ว เพราะกว่าจะนั่งคิดมันช้า แต่ขณะเดียวกัน NOKIA ขาย software ให้ซีเมนส์ แสดงว่า NOKIA ขึ้นไปอีกชั้นหนึ่งแล้ว พร้อมที่จะมี innovation ใหม่ นำไปอีกชั้นหนึ่ง นั่นคือการแข่งขันทางภาคธุรกิจ เหมือนกันครับ ของไทยก็แน่นอน ภาคเทคโนโลยีเราต้องคิดว่า วันนี้เราเป็นผู้ซื้อเทคโนโลยี ขณะเดียวกันเราก็พอคิดไหว แต่เรายังคิดในขั้น elementary ถ้าเราจะขึ้นสู่ secondary เราควรจะทำอย่างไร เราก็จะต้องทำในส่วนนั้น แต่ก็อย่าไปรอในส่วนนั้นจนไม่สามารถที่จะพัฒนาให้ทันสมัย (modernize)ตัวเอง เพราะเราจำเป็นต้อง modernize ตัวเองเหมือนกัน

            ในที่สุดแล้ว หลังจากที่เรารู้ปัญหา และเราร่วมกันทำแล้ว ผมอยากจะ อ้างคำกล่าว ( Quote ) สิ่งที่ เลขาธิการสหประชาชาติ ได้กล่าวสุนทรพจน์เมื่อคืนนี้ แก่ประเทศติมอร์ตะวันออก ว่า ไม่มีใครช่วย ติมอร์ตะวันออกได้ ต้องให้คนติมอร์ตะวันออก ต้องรวมเป็นเอกภาพ และต้องทำงานหนัก เพื่อให้ติมอร์ยืนอยู่บนลำแข้งตนเองได้ เช่นกันกับประเทศไทยเรา วันนี้พวกเราต้องสามัคคีกัน อย่าขัดแย้งกันเองในชาติมากนัก และ personal agenda ต้องมาทีหลัง ต้อง national agenda ก่อน และเราเชื่อว่าประเทศเรานั้น มีศักยภาพ เราปรับยุทธศาสตร์เราก็สู้เขาได้ ผมไม่ค่อยหนักใจ แต่สิ่งที่ผมหนักใจคือ unity ของคนในชาติ ขอขอบคุณทุกท่านที่มาร่วมกันแสดงความคิดเห็นวันนี้ มีอะไรที่จะแนะนำ ผมยินดี และผมพร้อมที่จะปรับเปลี่ยนทุกอย่างเพื่อให้ประเทศเราแข็งแรง ขอขอบคุณครับ
